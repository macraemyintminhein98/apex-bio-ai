# BioGen AI

BioGen AI is a web application designed to help users create compelling and personalized Instagram bios using the power of Artificial Intelligence. Users input keywords, their niche, and desired tone, and the AI generates several creative bio options.

## Features
- AI-powered bio generation
- Customizable inputs (keywords, niche, tone)
- Simple and intuitive user interface
- Secure payment processing for bio generations (Stripe integration planned)

## Technologies
- Frontend: React/Next.js (for a full application, currently a static HTML page demo)
- Backend: OpenAI API (GPT-3.5 Turbo)
- Payments: Stripe Checkout
- Hosting: Vercel

## Getting Started
To run this project locally (for the full Next.js application):
1. Clone the repository.
2. Install dependencies: `npm install` or `yarn install`.
3. Configure your OpenAI API key and Stripe secret key.
4. Run the development server: `npm run dev` or `yarn dev`.

For the static HTML demo, simply open `index.html` in your browser.
